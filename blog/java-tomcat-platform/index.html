使用BAE3.0搭建java-tomcat平台

2014年12月24日

BAE进入3.0版本，这里根据百度开放服务平台的帮助文档，说一说利用BAE搭建Java Web应用平台。

1.1 成为百度开发者

在百度开放平台http://developer.baidu.com/cloud/rt登录百度账号，选择“注册开发者”。

  • 填写开发者信息

1.2 创建应用

  • 进入管理控制台—>开发者服务管理，选择“创建工程”。勾选“使用BAE”，选择类型为“java-tomcat”。

  • 进入应用引擎，部署列表中工程“december24”的状态为“正常”。

  • 点击扩展服务，选择“添加新服务，MySQL”，选择“套餐一（免费）”。

创建应用后，可以使用版本控制工具部署代码，这里使用的是Git。

1.3版本管理

1.3.1 下载、安装msysGit

在Git for Windows http://msysgit.github.io/ 下载msysGit，这里下载的为Git-1.9.4-preview20140929.exe。运行下载的应用程序，采用默认配置即可。
如果配置不合适，可以重新运行下载的应用程序，进行配置。

1.3.2 下载、安装TortoiseGit

在Windows Shell Interface to Git https://code.google.com/p/tortoisegit/wiki/Download?tm=2 下载TortoiseGit，这里下载的为TortoiseGit-1.8.12.0-64bit.msi。运行下载的应用程序即可。

1.3.3 配置msysGit路径

  • 右键—>TortoiseGit，选择“Settings”。Git.exe Path即Git的安装路径。

1.3.4 克隆远程版本库

  • 右键—>Git clone。填写部署列表中的Git地址。

  • 验证百度账号用户名。考虑到用户名中可能包含中文，推荐使用邮箱地址作为用户名。

  • 本地版本库中出现以下目录：

1.4 Eclipse IDE for Java EE

利用Eclipse IDE for Java EE也可以完成上述1.3中的操作。

  • 进入Window—>Open Perspective，选择“Git”。点击左上角的“Clone a Git Repository…”按钮。填写部署列表中的Git地址和验证信息。

  • 填写本地版本库的路径。

  • 验证成功后，本地版本库中出现类似目录。
