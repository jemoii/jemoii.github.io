2014年11月21日

第一次

失败原因：忘记登录密码

安装系统时，分配了1GB内存、5GB硬盘空间，安装过程十分卡顿。安装完成后，重启登录时，一直提示密码错误，可能是安装设置root密码时，由于卡顿导致的错误。因为对操作还不熟悉，查找的方法也未能解决问题。
