Bitmaps

  • Bitmaps包含未操作命令；
  • setbit命令将bbit的某位置1或0，并返回该位的初始值；
  • getbit命令获取bbit某位的值；
  • bitcount命令返回bbit中值为1的位数
  • bitpos命令返回给定范围内第一个值为1或0的位，-1表示值不存在；
  • 可以添加参数设置bitpos命令的作用范围，范围是按字节划分，而不是比特位；
  • 结束范围缺省，默认为最后一个字节
